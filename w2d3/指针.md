# 指针

## 取内容符* 和取地址符&

&   取地址符    取变量的地址
*   取内容符    取缔之中存放的内容      在定义时表示声明指针，而不是取内容
(*	解引用)

两个例子：
如果int a = 6;
    int *p = &a;
那么
    *&a == ?    //a
    &*a == ?    //错误

## 指向数组名的指针与数组的区别
```c
char a[6] = "hello";
char *p = a;
//所以p == a吗
//验证
printf("%s\n", a);
printf("%s\n", p);
//数组一样
printf("%c\n", *p);
printf("%c\n", *a);
//偏移0一样
printf("%c\n", a[0]);
printf("%c\n", p[0]);
//直接命名一样（直接把数组a的名字换成p）
printf("%c\n", a[1]);
printf("%c\n", p[1]);
//这次换成看看偏移1
```
但是当：
p++
a++时，
a是地址常量，不能修改，p是指针变量可以修改

## 初始化
指针要存放普通变量的地址

定义指针时，一定要初始化，否则会产生野指针
```c
int a = 6;
int *p = &a;
// *p = a
*p = 666;

// 先定义后指向
int *p = NULL;
int a = 9;
p = &a;
```
## 存放数组首地址
int a[5] = {2, 5, 7};
int *p = a;             //此时p里面是a的首地址

## 指针运算
## 算数运算
char a[6] = "hello";
char *p = a;
此时：
p + n：结果向高地址方向移动n个数据单位
p++：指针向高地址移动1个数据单位
同一数组中：两个地址的差值 == 相差元素个数
```c
char a[6] = "hello";
char *p = a;
char *q = &a[2];
printf("%ld\n", q - p);         //  2
```
    
## 段错误产生的原因
1. 访问不存在的地址；
2. 访问系统保护的地址；
3. 访问只读的内存地址；
4. 空指针废弃；
5. 堆栈溢出；
6. 内存越界；

## 字节大小
指针字节大小是固定的
64 位系统：8字节
32 位系统：4字节
