# 指针2
## 指针修饰
### const
常量化：read only
#### 修饰普通变量
```c
int const a = 9;    //const int a = 9;
a++;    //×
```
但可以用指针改里面的内容
```c
int const a = 9;
int *p = &a;
*p = 99;
printf("%d\n", a);
```
#### 修饰指针
1. 修饰指向的内容，内容不可以被修改，但是可以改指针的指向
```c
int a = 9;
int const *p = &a;
*p = 99     //×
```
可以修改指针的指向
```c
int a = 9;
int const *p = &a;
int b = 99;
p = &b;
printf("%d\n", *p);
```
2. 修饰指向
```c
int a = 9;
int * const p = &a;
int b = 99;
p = &b;     //×
```
但此时可以修饰指向的内容
```c
int a = 9;
int * const p = &a;
*p = 99;
printf("%d\n", *p);
```

## 二级指针
一级指针：存放普通变量地址
二级指针：存放一级指针的地址
格式：
    数据类型  **指针变量名；
    int a = 6;
    int *p = &a;
    
获取a的值：     a       *p      **q
获取a的地址：   &a      p       *q
获取p的地址：           &p      q

## 指针和数组
直接访问：通过数组名
间接访问：通过指针

## 指针和二维数组
int a[2][3] = {2, 3, 4, 5, 6, 7};
a 首行地址
a + 1 次行地址
&a[i][j]
a:行地址 一行有多列 行地址级别高于列地址，可以加*降级处理
    *a 第一行第一列
    *a+1 第一行第二列
    *(a+1) 第二行第一列
