# 开辟堆区空间
## malloc
#include <stdio.h>
void *malloc{size_t size};
功能：开辟堆区空间
参数：字节大小
返回值：
    成功：堆区空间首地址
    失败：NULL

## free
#include <stdio.h>
void free(void *ptr);
功能：回收
参数：堆区空间首地址
返回值：无

# string函数
#include <string.h>
strlen strcpy strcat strcmp

## strcpy
#include <string.h>
char *strcpy(char *dest, const char *src);
功能：字符串的复制
参数：dest 目标地址   src 原字符串
返回值：dest 目标地址

char *strncpy(char *dest, const char *src, size_t n);
    n:复制的字符个数

## strcat
char *strcat(char *dest, const char *src);
功能：字符串的拼接
参数：把src拼接在dest后边
返回值：dest

## strcmp
#include <string.h>
int strcmp(const char *s1, const char *s2);
功能：字符串的比较
参数：s1 s2 字符串首地址
返回值：
    <0 s1<s2
    >0 s1>s2
    ==0 0
