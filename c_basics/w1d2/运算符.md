# 运算符

运算符
## 算术运算符
+	=	*	/	%	++	--
/ 整数相除，结果取整
++  自增（最高运算优先级）
## 逻辑运算符
&&	逻辑与	一假则假
||	逻辑或	一真则真
!	逻辑非	反转
### 截断法则
逻辑与：	前假则截断，返回假
逻辑或：	前真则截断，返回真
## 关系运算符
>	>=	<	<=	==	!=
## 位运算符
&	按位与	有0则0
|	按位或	有1则1
^	按位异或	相同为0，不同为1
~	按位取反	01互换
<<	右移	右移
>>	左移	左移
### 原码、反码、补码
计算机用补码存储、计算
正数三码相同
负数三码转换方式
原码	反码	补码
负数二进制	原码除符号位，其他全部取反	反码+1

## 三目运算符
表达式1? 表达式2: 表达式3;
类似if
判断表达式1是否成立，成立执行表达式2；否则执行表达式3
int a = 1, b = 2, c;
c = a > b ? a : b;
