# 输入输出
## 按格式输入输出
### printf
man 手册：产看命令、函数
man 3 printf

#include <stdio.h>
int printf(const char*format, ...)

功能：按格式终端输出
参数：不定长
返回值：输出元素个数
示例常用：printf(DAY"%d\n", a);
int		%d
char	%c
float		%f
double	%lf
字符串	%s
指针	%p
八进制	%o	%#o
十六进制	%x	%#x

### m.n
.n 保留n位小数
m 位宽
位宽的值必须大于数据的实际宽度
### scanf
man scanf

#include <stdio.h>
int scanf(const char*format, ...)

功能：按格式终端输入
参数：不定长
返回值：正确输入元素个数
第一个元素赋值失败，返回0
示例格式：
scanf("%d", &a);
## 按字符输入输出
### getchar

#include <stdio.h>
int getchar(void);

功能：在终端输入一个字符
参数：无
返回值：字符ASCII码

### putchar

#include <stdio.h>
int putchar(int c);

功能：终端输出一个字符
参数：字符ASCII码
返回值：字符ASCII码
垃圾字符回收
ASCII码表，空格'\n'都是字符
而scanf会产生垃圾字符，比如回车，这些字符会放在缓存区里
可以通过空格回收一个或多个空格、换行
# 分支语句
## if
基本结构
if(表达式1)
{
    表达式1成立，执行此代码
}
else if(表达式2)
{
    表达式1不成立，表达式2成立，执行此代码
}
……
else
{    
    上面的都不成立，执行此代码
}

可嵌套
只有一段代码时，{}可省略：if (a > b) printf("a > b");
## switch
swithc(表达式)
{
    case 常量1:
    break;
    case 常量2:
    break;
    case 常量3:
    break;
    case 常量4:
    break;
    default:
    break;
}

case后面只能跟着常量值：整型、字符型
case后面可以没有break，会从符合条件的分支开始知道整个分支结束，或遇见break
