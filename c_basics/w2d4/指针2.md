# 指针2
## 指针修饰
### const
常量化：read only
#### 修饰普通变量
```c
int const a = 9;    //const int a = 9;
a++;    //×
```
但可以用指针改里面的内容
```c
int const a = 9;
int *p = &a;
*p = 99;
printf("%d\n", a);
```
#### 修饰指针
1. 修饰指向的内容，内容不可以被修改，但是可以改指针的指向
```c
int a = 9;
int const *p = &a;
*p = 99     //×
```
可以修改指针的指向
```c
int a = 9;
int const *p = &a;
int b = 99;
p = &b;
printf("%d\n", *p);
```
2. 修饰指向
```c
int a = 9;
int * const p = &a;
int b = 99;
p = &b;     //×
```
但此时可以修饰指向的内容
```c
int a = 9;
int * const p = &a;
*p = 99;
printf("%d\n", *p);
```

## 大小端
大端；低地址存放高字节数据
小端：低地址存放低字节数据
int a = 0x12345678;
int 4字节==32位     1位十六进制转4位二进制
0x12345678

## 二级指针
一级指针：存放普通变量地址
二级指针：存放一级指针的地址
格式：
    数据类型  **指针变量名；
    int a = 6;
    int *p = &a;
    
获取a的值：     a       *p      **q
获取a的地址：   &a      p       *q
获取p的地址：           &p      q

## 指针和数组
直接访问：通过数组名
间接访问：通过指针

## 指针和一维数组
in a[5] = {2, 3, 4, 5, 6};
int a[5] = {2, 3, 4, 5, 6};

        地址        元素
a       &a[0]   2   a[0]    *a
a+1     &a[1]   3   a[1]    *(a+1)
a+2     &a[2]   4   a[2]    *(a+2)
a+3     &a[3]   5   a[3]    *(a+3)
a+4     &a[4]   6   a[4]    *(a+4)

int a[5] = {2, 3, 4, 5, 6};
int *p = a;

        地址        元素
&p[0]   p       2   *p      p[0]
&p[1]   p+1     3   *(p+1)  p[1]
&p[2]   p+2     4   *(p+2)  p[2]
&p[3]   p+3     5   *(p+3)  p[3]
&p[4]   p+4     6   *(p+4)  p[4]

## 指针和二维数组
int a[2][3] = {2, 3, 4, 5, 6, 7};
a 首行地址
a + 1 次行地址
&a[i][j]

    （首先列地址这个名词不存在，这个应该是数组a的第一个元素）
             （c语言里只有行指针和元素指针）
                            ⬇
a:行地址 一行有多列 行地址级别高于列地址，可以加*降级处理
    *a 第一行第一列
    *a+1 第一行第二列
    *(a+1) 第二行第一列

`                      地址            元素            `
`*a          a[0]    &a[0][0]    2   a[0][0]     *a[0]       **a`
`*a+1        a[0]+1  &a[0][1]    3   a[0][1]     *(a[0]+1)   *(*a+1)`
`*a+2        a[0]+2  &a[0][2]    4   a[0][2]     *(a[0]+2)   *(*a+2)`
`*(a+1)      a[1]    &a[1][0]    5   a[1][0]     *a[1]       *(*(a+1))`
`*(a+1)+1    a[1]+1  &a[1][1]    6   a[1][1]     *(a[1]+1)   *(*(a+1)+1)`
`*(a+1)+2    a[1]+2  &a[1][2]    7   a[1][2]     *(a[1]+2)   *(*(a+1)+2)`
