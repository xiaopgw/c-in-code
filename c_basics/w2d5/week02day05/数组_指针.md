# 数组/指针
## 数组指针
本质是指针，指向数组    行指针
格式：
    数据类型    (*指针变量名)[列数]：
```c
int a[2][3] = {2, 3, 4, 5, 6, 7};
int (*p)[3] = a;
```

```
                     地址           元素
*p          p[0]    &p[0][0]    2   p[0][0]     *p[0]       **p
*p+1        p[0]+1  &p[0][1]    3   p[0][1]     *(p[0]+1)   *(*p+1)
*p+2        p[0]+2  &p[0][2]    4   p[0][2]     *(p[0]+2)   *(*p+2)
*(p+1)      p[1]    &p[1][0]    5   p[1][0]     *(p[1])     *(*(p+1)+1)
*(p+1)+1    p[2]    &p[1][1]    6   p[1][1]     *(p[1]+1)   *(*(p+1)+1)
*(p+1)+2    p[3]    &p[1][2]    7   p[1][2]     *(p[1]+2)   *(*(p+1)+1)
```

## 指针数组
本质是数组，存放地址
格式：
    数据类型  *数组名[元素个数]
### 存放普通变量的地址
```c
int a = 10;
int b = 20;
int c = 30;
int *arr[3] = {&a, &b, &c};
```
arr[1]:     arr[1]==&b      *arr[1]==b
**(arr+1):  *(arr+1)==&b    **(arr+1)==b
### 存放二维数组的地址
存放二维数组的每一个第一列的地址
```c
int a[3][4] = {0};
int *arr[3] = {a[0], a[1], a[2]};
```
如果获取a[2][3]的地址：取arr的第三个元素(a的第三行第一列地址)+3
### 存放字符串常量区首地址
```c
char *arr[3] = {"hello", "world", "hq"};
```
如果打印world：取arr第二个元素

# 函数
一次编写，多次调用
一段实现某个功能的代码段
格式：

## 数据类型 函数名(形式参数1, 形式参数2)
{
    实现某个功能的代码段
    return 常量值或变量;
}
子函数也可以定义在主函数下边，需要提前声明。
## 函数传参
### 值传递
单向传递
```c
void fun(int a)
{
    a++;
}
int main()
{
    int a = 1;
    fun(a);
    printf("%d\n", a);  \\a = 1
}
```

### 地址传递
双向传递
```c
void fun(int *a)
{
    (*a)++;
}
int main()
{
    int a = 1;
    fun(a);
    printf("%d\n", a);  \\a = 2
}
```
