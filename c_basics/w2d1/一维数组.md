# 数组
由一个或多个相同类型的数据组成的集合
数据类型相同
内存连续
## 一维数组
定义
格式：
数据类型 数组名[元素个数]
int a[5] = {2, 3, 4, 5, 6};
char b[5] = {'q', 'w', 'e'};
访问
a[n]是表示偏移量

## 数组名
arr	数组首地址
&arr	数组指针（整个数组的地址）
*arr	数组首元素的值（等价于arr[0]）

## 字节大小
len = sizeof(arr) / sizeof(arr[0])

## 初始化
全部初始化
部分初始化：未初始化部分为0
未初始化：随机值

## 内存分配
连续的内存

## 清零函数
bzero
#include <strings.h>
void bzero(void *s, size_t n);
功能：数组元素清零
参数：s:数组首地址	n:字节大小
返回值：无
void 空
void* 任意类型指针
memset
#include <string.h>
void* memset(void *s, int c, size_t n);
功能：数组元素清零
参数：s：数组首地址		n：字节大小		c 要设置的值
返回值：数组首地址
